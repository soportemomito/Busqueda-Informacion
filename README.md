# SoyMomo ST System

Herramienta de búsqueda para el equipo de soporte de SoyMomo. Consolida en una sola consulta información de Chatwoot, Shopify, Bsale y Google Drive.

## Qué hace

- Busca por nombre, teléfono, email, número de orden o ID/IMEI del dispositivo.
- Muestra conversaciones de Chatwoot, pedidos de Shopify, documentos de Bsale y archivos de Drive relacionados con el cliente.
- Extrae identificadores del dispositivo (IMEI, ICCID/SIM, serial) de los mensajes y los cruza con otros tickets históricos.
- Se integra como Dashboard App dentro de Chatwoot: al abrir una conversación carga automáticamente la búsqueda del cliente.
